# Open Data AI Analytics

## 📌 Project name and goal

**Open Data AI Analytics** — це навчальний проєкт для аналізу відкритих даних з використанням Python, Pandas, Jupyter Notebook та методів аналізу даних.
Мета проєкту — дослідити відкритий набір даних, виконати очищення, аналіз та отримати корисні висновки на основі статистики та візуалізації.

Проєкт створений для практики роботи з open data та інструментами data analytics.

---

## 📊 Open data source

Джерело відкритих даних:
https://data.gov.ua/

Dataset використовується з порталу відкритих даних України.
Конкретний набір даних буде доданий у папку `data/raw`.

---


## ❓ Questions / Hypotheses for analysis (edited in branch 1 and branch 2)


1. Чи змінюються показники у даних з часом (тренд зростання або спадання)?
2. Чи існує залежність між основними параметрами в наборі даних?
3. Які значення є аномальними (outliers) і як вони впливають на результат аналізу?
